# Quickstart

The fastest path: add HyperMemory as an MCP server in Claude Desktop or Claude Code with OAuth.

## 1. Sign up

Go to [app.hypermemory.io/signup](https://app.hypermemory.io/signup) and create an account with GitHub.

## 2. Add the MCP server

Open your Claude Desktop config file:

| OS | Path |
|---|---|
| macOS | `~/Library/Application Support/Claude/claude_desktop_config.json` |
| Windows | `%APPDATA%\Claude\claude_desktop_config.json` |
| Linux | `~/.config/Claude/claude_desktop_config.json` |

Add HyperMemory:

```json
{
  "mcpServers": {
    "hypermemory": {
      "url": "https://api.hypermemory.io/mcp"
    }
  }
}
```

Restart Claude Desktop. It will prompt you to authorize with your HyperMemory account via OAuth.

<Tip>
**Claude Code** — run `claude mcp add hypermemory --transport http https://api.hypermemory.io/mcp` instead.
</Tip>

## 3. Verify

Ask Claude:

> What memory tools do you have?

You should see tools starting with `hm_` — `hm_store`, `hm_recall`, `hm_find_related`, and more.

## 4. Store your first memory

> Remember that our project deadline is March 31st, 2026.

Claude calls `hm_store` with a key, description, and optional structured data. The server triggers background enrichment to expand the description, extract entities, and create relationships automatically.

## 5. Recall it

> When is our project deadline?

Claude calls `hm_recall`, which runs a hybrid search combining BM25 full-text matching, vector similarity, and regex to find the best results.
