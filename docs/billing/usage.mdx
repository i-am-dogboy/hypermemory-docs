# Usage

## What counts as a query

Every `tools/call` request to the MCP endpoint counts as one query, regardless of which tool is called. `tools/list` does not count.

REST API calls to `/api/v1/memory/*` endpoints also count as queries.

## Viewing usage

Go to [app.hypermemory.io/usage](https://app.hypermemory.io/usage) to see:

- Current period query count vs plan limit
- Daily usage breakdown

## Monthly query limits

| Plan | Queries/mo |
|---|---|
| Free | 2,000 |
| Basic | 10,000 |
| Pro | 200,000 |
| Business | 500,000 |
| Enterprise | Custom |

## Overages

- **Free:** hard limit — queries stop at 2,000. No overages.
- **Basic:** $0.80 per 1,000 additional queries.
- **Pro:** $0.08 per 1,000 additional queries.
- **Business:** $0.10 per 1,000 additional queries.
- **Enterprise:** custom limits.

## REST API usage endpoints

| Endpoint | Method | Description |
|---|---|---|
| `/v1/usage` | GET | Current period usage |
| `/v1/usage/history` | GET | Daily usage snapshots |

Authenticate with a Bearer API key or OAuth token.
