## File storage limits

| Plan | Storage |
|---|---|
| Free | Disabled |
| Basic | Disabled |
| Pro | 1 GB |
| Business | 100 GB |
| Enterprise | 1 TB |

Files are stored in S3 (Scaleway). Text files are decoded directly and summarized by an LLM. Binary, office, and image files are processed through LlamaParse extraction before summarization. Each uploaded file creates a graph node with the AI-generated summary.

## Graph limits

Each plan has a maximum number of memory graphs (separate knowledge bases):

| Plan | Graphs |
|---|---|
| Free / Basic | 1 |
| Pro | 4 |
| Business | 20 |
| Enterprise | Unlimited |

On plan downgrade, excess graphs become **frozen** (read-only). Upgrade to unfreeze.
