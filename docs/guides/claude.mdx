# Claude Desktop & Code

## Claude Desktop

### With OAuth (recommended)

Add to `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "hypermemory": {
      "url": "https://api.hypermemory.io/mcp"
    }
  }
}
```

Restart Claude Desktop. It will open a browser window for OAuth sign-in with GitHub (via Supabase OAuth 2.1 + PKCE).

Config file location:

| OS | Path |
|---|---|
| macOS | `~/Library/Application Support/Claude/claude_desktop_config.json` |
| Windows | `%APPDATA%\Claude\claude_desktop_config.json` |
| Linux | `~/.config/Claude/claude_desktop_config.json` |

### With API key

```json
{
  "mcpServers": {
    "hypermemory": {
      "url": "https://api.hypermemory.io/mcp",
      "headers": {
        "Authorization": "Bearer hm_YOUR_KEY_HERE"
      }
    }
  }
}
```

Generate keys at **Dashboard > Settings > API Keys** at [app.hypermemory.io](https://app.hypermemory.io). Keys use the format `hm_` + 64 hex characters and are bound to a specific graph.

### Self-hosted (stdio)

```json
{
  "mcpServers": {
    "hypergraph-memory": {
      "command": "python3",
      "args": ["/path/to/hypergraph_memory/src/server.py"],
      "env": {
        "HYPERGRAPH_MEMORY_FILE": "/path/to/memory.json"
      }
    }
  }
}
```

## Claude Code

### With OAuth

```bash
claude mcp add hypermemory --transport http https://api.hypermemory.io/mcp
```

### With API key

```bash
claude mcp add hypermemory \
  --transport http \
  --header "Authorization: Bearer hm_YOUR_KEY_HERE" \
  https://api.hypermemory.io/mcp
```

### Self-hosted (stdio)

```bash
claude mcp add hypergraph-memory \
  --transport stdio \
  -- python3 /path/to/hypergraph_memory/src/server.py
```

## Verify

Ask Claude: **"What memory tools do you have?"**

You should see 11 tools prefixed with `hm_` — `hm_store`, `hm_recall`, `hm_find_related`, `hm_get_overview`, `hm_ingest`, and more.

<Info>
If using the self-hosted server, you will see `memory_*` tools instead (e.g. `memory_store`, `memory_recall`). The self-hosted open-source server uses different tool names from the hosted service.
</Info>

## Usage tips

- **Store:** "Remember that our deadline is March 31st" triggers `hm_store`
- **Recall:** "What do you know about deadlines?" triggers `hm_recall`
- **Ingest:** Paste a large block of text and ask Claude to ingest it — triggers `hm_ingest` to decompose it into graph entities in one LLM pass
- Be explicit — Claude may not use memory tools unless prompted
