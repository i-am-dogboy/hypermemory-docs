# CrewAI

CrewAI agents can use HyperMemory through MCP. The pattern: connect to the hosted MCP server (or run self-hosted), then call tools via an MCP client in your CrewAI tool wrappers.

## Option 1: Hosted (API key)

Set your API key:

```bash
export HYPERMEMORY_API_KEY=hm_YOUR_KEY_HERE
```

Create tool wrappers that call the hosted MCP endpoint:

```python
import os
import httpx

MCP_URL = "https://api.hypermemory.io/mcp"
API_KEY = os.environ["HYPERMEMORY_API_KEY"]

def call_mcp(tool_name: str, arguments: dict) -> dict:
    response = httpx.post(
        MCP_URL,
        json={
            "jsonrpc": "2.0",
            "id": 1,
            "method": "tools/call",
            "params": {"name": tool_name, "arguments": arguments},
        },
        headers={
            "Authorization": f"Bearer {API_KEY}",
            "Content-Type": "application/json",
        },
        timeout=30.0,
    )
    return response.json().get("result", {})
```

Then wrap `call_mcp` as a CrewAI `BaseTool` and attach it to your agents.

## Option 2: Self-hosted (stdio)

If running from the [open-source repo](/connect/self-hosted), start `python src/server.py` as a sidecar process or use the HTTP wrapper at `deploy/mcp-server/mcp_http_server.py`, then point your `call_mcp` function at `http://localhost:8080`.

## Key tools for agents

| Tool | Purpose |
|---|---|
| `hm_store` | Persist a fact, decision, or preference |
| `hm_recall` | Hybrid search (BM25 + vector + regex) |
| `hm_find_related` | Traverse from a node |
| `hm_get_overview` | Graph stats and top nodes |
| `hm_ingest` | Decompose dense text into graph entities |
| `hm_forget` | Remove outdated info |

See [API Reference](/api-reference/overview) for all 11 tools.
